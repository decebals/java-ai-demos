Pre-requisite
-------------------
Java 21+

How to build
-------------------
`./mvnw clean package`

How to run
-------------------
`./mvnw -pl <module-name> spring-boot:run` (e.g. `./mvnw -pl langchain4j-spring-demo spring-boot:run`)
